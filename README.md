There is no official Azure certification dedicated solely to Azure Kubernetes Service (AKS).

If you want to prove AKS mastery:

✔️ AZ 305 - Azure Solutions Architect Expert
- Designing compute solutions  
- Designing containerized workloads  
- Networking and security for AKS  

✔️ AZ 400 - DevOps Engineer Expert
- CI/CD for AKS  
- GitOps  
- Container build pipelines  

✔️ AZ 104 - Azure Administrator Associate
- Deploying and managing AKS clusters  
- Scaling and updating nodes  
- Integrating ACR  

✔️ Complete the AKS learning paths on Microsoft Learn
- [Introduction to Kubernetes on Azure](https://learn.microsoft.com/en-us/training/paths/intro-to-kubernetes-on-azure/)  
- [Baseline Architecture AKS Cluster](https://learn.microsoft.com/en-us/azure/architecture/reference-architectures/containers/aks/baseline-aks?toc=%2Fazure%2Faks%2Ftoc.json&bc=%2Fazure%2Faks%2Fbreadcrumb%2Ftoc.json) 
- AKS scalability and operations
- [Develop and deploy AKS applications](https://learn.microsoft.com/en-us/training/paths/develop-deploy-applications-kubernetes/)

✔️ Optionally add CNCF certifications:
- CKA (Kubernetes Administrator)  
- CKAD (Kubernetes Application Developer)
